## Como comparar depois

```bash
ssh wp-napoleon 'wp --path=$HOME/domains/labresumos.com.br/public_html plugin list --format=table'
ssh wp-napoleon 'wp --path=$HOME/domains/labresumos.com.br/public_html cron event list --fields=hook,schedule'
```
Se algo aqui mudar sem passar por este repo, é sinal de deploy manual fora do fluxo —
investigar antes de continuar o roadmap.
