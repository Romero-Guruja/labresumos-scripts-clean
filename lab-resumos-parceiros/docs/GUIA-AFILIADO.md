# Guia do Parceiro
## Programa de Parceiros Lab Resumos

---

## Bem-vindo!

Como parceiro Lab Resumos, você ganha comissões divulgando nossos cursos.

**Página de cadastro:** `seusite.com/seja-parceiro/`

**Seu painel (após aprovação):** `seusite.com/meu-painel-parceiro/`

---

## Como se Cadastrar

1. Acesse a página de cadastro
2. Preencha seus dados:
   - Nome completo
   - Email
   - Senha (se não tiver conta)
   - Dados de pagamento (PIX)
   - Código do cupom desejado (opcional)
3. Aceite os termos
4. Envie o cadastro
5. Aguarde aprovação (você receberá um email)

> 💡 Se já tem conta no site, faça login primeiro para preencher automaticamente seus dados.

## 3. Comissões

### 3.1 Cálculo

A comissão é sempre sobre o **valor pago** pelo cliente (após descontos).

**Exemplo:**
- Curso: R$ 500
- Desconto cupom: 10% = R$ 50
- Cliente paga: R$ 450
- Sua comissão (10%): **R$ 45**

### 3.2 Status das Comissões

| Status | Significado |
|--------|-------------|
| Pendente | Pedido feito, aguardando pagamento |
| Aprovada | Pedido pago, comissão confirmada |
| Paga | Você já recebeu |
| Cancelada | Pedido cancelado ou reembolsado |

### 3.3 Quando Recebo?

1. **Fechamento:** Dia 1 de cada mês
2. **Mínimo:** R$ 200 para sacar
3. **NF:** Você envia após o fechamento
4. **Pagamento:** PIX em até 5 dias úteis após aprovação da NF

> 💡 Não atingiu R$ 200? Seu saldo acumula para o próximo mês.

## 4. Conflito Guruja

Se o cliente for aluno Guruja e tiver desconto próprio:

- O sistema aplica o **maior desconto**
- Você ainda ganha comissão sobre o valor pago

## 6. Dúvidas Frequentes

**Posso usar meu cupom nas minhas compras?**
Depende da sua configuração. Verifique no seu painel ou com o suporte.

**Quanto tempo o cookie dura?**
60 dias. Se o cliente clicar no seu link hoje, você ganha a comissão se ele comprar em até 60 dias.

**E se o cliente usar cupom E link?**
O cupom tem prioridade. Você ganha a comissão de cupom (10%).

**Como sei se uma venda foi minha?**
Confira na aba **Vendas** do seu painel. Todas as vendas atribuídas aparecem lá.

**Preciso emitir Nota Fiscal?**
Sim, para receber o pagamento você precisa enviar NF de prestação de serviços.

**Qual o valor mínimo para saque?**
R$ 200. Se não atingir, o saldo acumula.
